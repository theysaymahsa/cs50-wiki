CS50 Wiki

A simple Wiki project built with Django as part of the CS50 Web course.

Project Overview

This project allows users to view, create, and edit encyclopedia entries. It is built using Django and includes multiple apps such as encyclopedia and entries.

Technologies

Python 3, Django, HTML/CSS, SQLite (default database)

Setup & Run Locally

Clone the repository, navigate to the project folder, create a virtual environment if you want, install dependencies (or Django manually), run migrations, and start the development server. Then open your browser at http://127.0.0.1:8000/
 to see the project.

Notes

This project is backend-based (Django), so GitHub Pages cannot host it live. All code and structure can be viewed directly on GitHub.
